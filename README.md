# C++ Modules 00–09 (Hive Helsinki / 42)

This repository contains my solutions for **C++ Modules 00 → 09** from the Hive Helsinki (42) curriculum. The modules build a solid foundation in modern C++ fundamentals: OOP, memory management, inheritance, polymorphism, exceptions, templates, and STL.

**Goal:** strengthen core C++ skills and write clean, maintainable, correct code.

---

## Modules Overview

- **Module 00**: Basics — namespaces, classes, member functions, I/O, Makefile
- **Module 01**: Memory & references — new/delete, pointers vs references, file streams
- **Module 02**: Orthodox Canonical Form — copy ctor, assignment operator, fixed-point
- **Module 03**: Inheritance — base/derived classes, constructors/destructors
- **Module 04**: Polymorphism & abstract classes — virtual functions, interfaces
- **Module 05**: Exceptions — try/catch, custom exceptions, robust error handling
- **Module 06**: Type conversion — casts, serialization basics
- **Module 07**: Templates — function/class templates
- **Module 08**: STL — containers, iterators, algorithms, complexity
- **Module 09**: Practical STL projects — parsing, containers/algorithms in mini-app style

> Note: exact exercise names may vary by subject version, but the core topics remain consistent.

## How to use

Each module is organized into exercises (`ex00`, `ex01`, ...).  
To build and run an exercise:

```bash
cd cpp_XX/exYY
make
./<executable_name>
```
> Tested on Linux (Ubuntu) with `c++` and standard 42/Hive flags (`-Wall -Wextra -Werror`).
